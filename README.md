## ¡Hola! Soy Felipe 👋

Soy **Growth Product Manager**: encuentro dónde se atasca el crecimiento de un producto y lo desbloqueo con datos, experimentos y foco.

Con una diferencia: no me quedo en el roadmap. Cuando una hipótesis necesita software para validarse, **lo construyo yo mismo** — estos repos son la prueba.

- 📈 **Growth con método** — hipótesis, experimento, métrica. Nada de vanity metrics.
- 📊 **De datos a decisiones** — tableros ejecutivos, Gantt y PERT que se leen en 30 segundos.
- 🛠️ **PM que construye** — prototipo, valido y despliego sin esperar el sprint de nadie.
- 🌅 **Con identidad** — el producto puede ser útil *y* tener alma. Pregúntale al chigüiro de más abajo.

## 🧭 Trayectoria en cuatro actos

1. **Automatizar** — scripts de Excel, SQL y Python que devolvían horas: aprendí que el crecimiento empieza quitando fricción.
2. **Construir** — SPAs y sitios en Vue y JavaScript, para entender el producto desde el código.
3. **Producto end-to-end** — AgileFlow, FlowBoard y Pomodoro Llanero: detecté el problema, definí el alcance y lo lancé.
4. **Growth** — hoy uno [datos, producto y ejecución](https://github.com/castellanosfelipe) para hacer crecer lo que toco — documentado como brief de producto.
